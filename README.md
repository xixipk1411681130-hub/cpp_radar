# cpp_radar
雷达信号处理，cpp实现


线性调频信号，生成一个采样点个数为1668的脉冲压缩前的LFM信号，完成整个信号处理流程，使用cpp使用，搭建完整的工程并进行相关运行测试。具备以下一些要求
-使用多线程处理，在linux环境下，最多有16个核，要充分利用硬件资源。
-生成造数模块，按照正常一个脉冲500us，一直循环，造数和处理。
-进行脉冲压缩处理。
-进行MTD处理，使用vsipl库，进行16点滑动处理。
-每个通道进行cacfar处理，然后通道选大，得到输出结果和多普勒通道号。
-进行仿真验证和长时间运行测试，保证数据的输出顺序和输入顺序一致，不能出现错乱。
-使用软件化雷达思想，进行模块与功能块解耦，保证代码质量，并进行完整的说明和注释，尽量详细。
